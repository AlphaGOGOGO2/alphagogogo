## 알파뮤직 AI란?

알파뮤직 AI는 비공식 Suno API(sunoapi.org)를 활용하여 AI 음악을 대량으로 생성할 수 있는 데스크톱 프로그램입니다. ChatGPT와 Gemini 등 여러 AI 모델을 연동하여 프롬프트를 자동 생성하고, 한 번에 최대 15곡까지 음악을 만들 수 있는 강력한 도구입니다.

### 주요 특징
- 수노 API를 통한 음악 대량 생성 (최대 15곡)
- ChatGPT, Gemini 연동으로 프롬프트 자동 생성
- 가사 작성 및 편집 기능
- 음악 스타일, 장르, BPM 등 세부 설정 가능
- 생성된 음악 라이브러리 관리

## 알파뮤직 AI 설치 방법

### 1단계: 설치 파일 다운로드 및 실행

![설치1.png](/images/blog/1763636168601-설치1.png)

설치 프로그램을 다운로드하여 실행하면 설치 마법사가 시작됩니다. "Welcome to 알파뮤직 AI Setup" 화면에서 "Next" 버튼을 클릭하여 설치를 진행합니다.

### 2단계: 설치 경로 선택

![설치파일2경로설정.png](/images/blog/1763636182000-설치파일2경로설정.png)

설치 위치를 선택하는 화면입니다. 기본 경로는 `C:\Program Files\AlphaGOGOGO2\알파뮤직 AI` 입니다. 원하는 경로로 변경하거나 기본 경로를 사용한 후 "Install" 버튼을 클릭합니다.

### 3단계: 설치 완료


설치가 완료되면 데스크톱에 "알파뮤직 AI" 아이콘이 생성됩니다. "Finish" 버튼을 클릭하여 설치를 마무리합니다.

![설치완료아이콘생성.png](/images/blog/1763636199357-설치완료아이콘생성.png)

데스크톱에 생성된 알파뮤직 AI 아이콘을 확인할 수 있습니다.

## 프로그램 초기 설정

### API 키 설정

![실행화면.png](/images/blog/1763636250062-실행화면.png)

![api설정창.png](/images/blog/1763636253584-api설정창.png)

프로그램을 처음 실행하면 API 설정이 필요합니다.

1. **프롬프트 생성 모델 선택**: ChatGPT 또는 Gemini 중 선택
2. **OpenAI API Key**: ChatGPT를 사용할 경우 입력
3. **Suno API Key**: 수노 API 키 입력 (sunoapi.org에서 발급)

각 API 키 옆의 "테스트" 버튼을 클릭하여 정상 작동하는지 확인한 후 "저장" 버튼을 클릭합니다.

![테스트.png](/images/blog/1763636269392-테스트.png)

API 키가 정상적으로 작동하면 체크 표시가 나타납니다.

## 알파뮤직 AI 실행 화면


![실행화면.png](/images/blog/1763636281509-실행화면.png)

프로그램 메인 화면입니다. 우측 상단에는 다양한 템플릿과 가이드가 있으며, 좌측에서 음악 생성 설정을 할 수 있습니다.

### 메인 기능 구성
- **음악 설명**: 만들고 싶은 음악에 대한 설명 입력
- **고급 설정**: 장르, 세부 장르, 음악 타입, 보컬 타입 등 설정
- **생성 횟수**: 한 번에 생성할 음악 개수 (최대 15곡)
- **저장 경로**: 생성된 음악 파일 저장 위치

## 음악 생성 과정

### 1단계: 음악 설정

![세팅.png](/images/blog/1763636291654-세팅.png)

음악 생성 전 기본 설정을 합니다:
- 음악에 대한 설명 입력
- 고급 설정에서 세부 옵션 조정
- 생성 횟수 선택 (1곡 요청 = 2곡 생성)
- 저장 경로 설정

### 2단계: 고급 옵션 설정


![고급옵션1.png](/images/blog/1763636304257-고급옵션1.png)
![고급옵션2.png](/images/blog/1763636306859-고급옵션2.png)
고급 옵션에서는 다음을 설정할 수 있습니다:
- **장르**: 원하는 음악 장르 선택
- **세부 장르**: 더 구체적인 장르 세분화
- **음악 타입**: 보컬(가사 포함) 또는 인스트루멘탈
- **보컬 타입**: 보컬 스타일 선택
- **보컬 성별**: 남성, 여성 또는 혼성
- **템포**: 60~180 BPM 범위에서 조절
- **분위기**: 음악의 전반적인 느낌 설정
- **악기 구성**: 사용할 악기 선택


추가 설정 옵션:
- **가사 언어**: 가사 작성 언어 선택 (영어, 한국어 등)
- **AI 모델**: Suno V5 등 모델 선택
- **제외할 요소**: 특정 요소 제외 설정
- **추가 요청사항**: 기타 원하는 스타일 입력
- **생성 횟수**: 1회 요청당 2곡 생성 (최대 15곡)
- **저장 경로**: 음악 파일 저장 위치 지정

### 3단계: 프롬프트 생성

![프롬프트1.png](/images/blog/1763636318044-프롬프트1.png)
"프롬프트 생성" 버튼을 클릭하면 AI가 설정에 맞는 음악 프롬프트를 자동으로 생성합니다. 생성된 프롬프트에는 다음 정보가 포함됩니다:

- **제목**: 음악 제목 (예: "Quiet Rivers, Soft Skies")
- **스타일**: 음악 스타일 설명
- **메타데이터 프롬프트**: 보컬 스타일, 장르, 템포, 악기 편성, 분위기 등 상세 정보

![프롬프트2가사.png](/images/blog/1763636328419-프롬프트2가사.png)

가사가 포함된 경우, 영어로 작성된 가사도 함께 생성됩니다:
- **Intro**: 도입부 가사
- **Verse**: 절 가사
- **Chorus**: 후렴구
- **Bridge**: 브릿지 섹션 등

### 4단계: 음악 생성 시작

![음악생성시작.png](/images/blog/1763636339246-음악생성시작.png)


모든 설정이 완료되면 "음악 생성" 버튼을 클릭합니다.

![요청중.png](/images/blog/1763636346543-요청중.png)

음악 생성이 시작되면 우측 패널의 "생성 상태"에서 진행 상황을 실시간으로 확인할 수 있습니다. "요청 #1" 항목에서 생성 진행률을 볼 수 있습니다.

### 5단계: 생성 완료 및 음악 재생

![생성완료.png](/images/blog/1763636354988-생성완료.png)

음악 생성이 완료되면 우측 음악 플레이어에서 바로 재생할 수 있습니다. 

- 생성된 음악 제목 확인 (예: "Quiet Rivers, Soft Skies")
- 재생/일시정지 컨트롤
- 볼륨 조절
- 음악 라이브러리에서 전체 목록 확인

생성된 음악 파일은 지정한 저장 경로에 자동으로 저장됩니다.

## 알파뮤직 AI 활용 팁

### 1. 프롬프트 최적화
- 구체적인 음악 설명을 입력할수록 원하는 결과물을 얻을 가능성이 높아집니다
- 고급 옵션을 세밀하게 조정하면 더욱 정교한 음악을 만들 수 있습니다

### 2. 대량 생성 전략
- 한 번에 최대 15곡까지 생성 가능
- 유사한 스타일의 음악을 여러 버전으로 만들어 비교할 수 있습니다

### 3. API 크레딧 관리
- Suno API는 크레딧 기반으로 작동합니다
- 생성 전 크레딧 잔여량을 확인하세요

### 4. 다양한 AI 모델 활용
- ChatGPT와 Gemini를 번갈아 사용하여 다양한 프롬프트 스타일을 경험해보세요

## 주의사항

- 생성된 음악의 저작권 및 사용권은 Suno API 이용 약관을 따릅니다
- 비공식 API 사용이므로 서비스 안정성이 변동될 수 있습니다
- API 키는 안전하게 보관하고 공유하지 마세요
