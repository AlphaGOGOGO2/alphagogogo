# 러버블 dev ai 사용 후기: 코딩 초보도 앱 만들 수 있을까?![블로그 이미지](/blog-images/e4ba33fe-8ae7-44ca-95e4-74491980b435.png)최근 AI 개발 도구들이 정말 많이 나오고 있죠. 그중에서도 러버블(Lovable) dev ai가 특히 주목받고 있어서, 직접 사용해보고 솔직한 후기를 남겨보려고 합니다. 사실 저도 코딩에 대해서는 완전 초보 수준이라 과연 이런 AI 도구로 제대로 된 앱을 만들 수 있을까 반신반의했거든요.

## 러버블 dev ai가 뭔가요?

러버블 dev ai는 자연어로 대화하듯 명령을 내리면 실제 작동하는 웹 애플리케이션을 만들어주는 AI 개발 도구입니다. 간단히 말해서 "이런 기능이 있는 앱을 만들어줘"라고 말하면, AI가 알아서 코드를 작성해서 실제 앱을 만들어주는 거죠.

기존의 코딩 AI들과 다른 점은 단순히 코드만 생성해주는 게 아니라, 실제로 배포 가능한 완성된 애플리케이션을 만들어준다는 것입니다. 정말 혁신적이지 않나요?

## 실제 사용해본 경험담

### 첫 인상과 인터페이스

처음 러버블에 접속했을 때 느낌은... 음, 생각보다 심플하더라고요. 복잡한 설정이나 어려운 용어들로 가득할 줄 알았는데, 그냥 채팅하듯이 대화할 수 있는 인터페이스였습니다. 

"간단한 할 일 관리 앱을 만들어줘"라고 입력했더니, 정말로 몇 분 만에 실제 작동하는 앱이 나오더군요. 처음에는 믿기지 않았어요.

### 만들어본 프로젝트들

#### 1. 할 일 관리 앱

가장 먼저 시도해본 건 간단한 투두 리스트 앱이었습니다. 할 일 추가, 완료 체크, 삭제 기능이 있는 기본적인 앱을 요청했는데:

• 깔끔한 UI 디자인
• 반응형 웹 디자인으로 모바일에서도 잘 보임
• 실시간으로 데이터 저장 및 불러오기
• 직관적인 사용자 경험

이 모든 게 제가 아무것도 모르는 상태에서 만들어진 거라니, 정말 놀라웠습니다.

#### 2. 간단한 일기 앱

두 번째로는 조금 더 복잡한 일기 앱을 시도해봤어요. "날짜별로 일기를 작성하고 저장할 수 있는 앱을 만들어줘"라고 했더니:

• 달력 형태의 인터페이스
• 날짜 선택 기능
• 텍스트 에디터 기능
• 검색 기능까지

개인적으로 이 프로젝트에서 가장 인상 깊었던 건, 제가 추가로 "검색 기능도 넣어줘"라고 요청했을 때 기존 앱을 완전히 새로 만드는 게 아니라 기능만 추가해준 점이었어요.

## 장점들

### 1. 진입 장벽이 낮다

가장 큰 장점은 역시 코딩 지식이 없어도 된다는 점입니다. 저처럼 HTML, CSS, JavaScript 같은 기본 언어도 제대로 모르는 사람도 실제 작동하는 앱을 만들 수 있어요.

### 2. 빠른 프로토타이핑

아이디어가 있을 때 빠르게 구현해볼 수 있다는 게 정말 좋더라고요. 예전에는 개발자를 찾거나 직접 공부해야 했는데, 이제는 몇 분 만에 프로토타입을 만들어볼 수 있으니까요.

### 3. 실시간 수정 가능

코드를 직접 수정할 필요 없이 "이 부분을 이렇게 바꿔줘"라고 말하면 바로 반영되는 것도 편리했습니다. 마치 디자이너와 실시간으로 협업하는 느낌이에요.

### 4. 학습 도구로서의 가치

코딩을 배우고 싶은 사람들에게는 좋은 학습 도구가 될 것 같아요. AI가 생성한 코드를 보면서 "아, 이런 식으로 구현하는구나" 하고 배울 수 있거든요.

## 아쉬운 점들

물론 완벽하지는 않았어요. 사용하면서 느낀 몇 가지 아쉬운 점들이 있었습니다.

### 1. 복잡한 기능 구현의 한계

간단한 CRUD(생성, 읽기, 수정, 삭제) 기능은 정말 잘 만들어주는데, 복잡한 비즈니스 로직이 필요한 기능들은 아직 한계가 있더라고요. 예를 들어, "사용자 권한에 따라 다른 기능을 보여줘"같은 요청은 완벽하게 구현되지 않았어요.

### 2. 디자인의 획일성

AI가 만들어주는 디자인들이 비슷비슷한 느낌이 있어요. 물론 깔끔하고 modern한 디자인이지만, 개성 있는 디자인을 원한다면 별도로 요청해야 합니다.

### 3. 데이터베이스 연동 이슈

복잡한 데이터베이스 구조나 외부 API 연동 같은 부분에서는 가끔 오류가 발생하더라고요. 이런 부분은 아직 개선이 필요해 보입니다.

## 비용 대비 효과는?

러버블 dev ai의 가격 정책을 보면, 무료 플랜도 있지만 제한이 있고, 본격적으로 사용하려면 유료 플랜을 이용해야 해요. 

개인적으로는 웹 개발을 외주로 맡겼을 때의 비용을 생각하면 충분히 합리적이라고 생각합니다. 간단한 웹앱 하나 만드는데 보통 수십만 원에서 수백만 원이 들잖아요. 그런데 러버블로는 월 구독료만으로 여러 개의 앱을 만들어볼 수 있으니까요.

## 어떤 사람들에게 추천할까?

### 적극 추천하는 경우

• **스타트업 창업자**: 아이디어를 빠르게 검증하고 싶은 분들
• **마케터나 기획자**: 개발팀 없이도 간단한 도구를 만들고 싶은 분들  
• **코딩 입문자**: 프로그래밍을 배우면서 실습해보고 싶은 분들
• **개인 프로젝트**: 취미로 간단한 웹앱을 만들어보고 싶은 분들

### 신중하게 고려해야 하는 경우

• **복잡한 엔터프라이즈급 애플리케이션**이 필요한 경우
• **고도의 커스터마이징**이 필요한 프로젝트
• **실시간 성능이 중요한 애플리케이션**

## 앞으로의 전망

러버블 dev ai는 정말 빠르게 발전하고 있어요. 제가 사용하기 시작한 지 몇 주 만에도 새로운 기능들이 추가되고 기존 기능들이 개선되는 걸 체감할 수 있었거든요.

특히 AI 기술이 계속 발전하면서 더 복잡한 애플리케이션도 만들 수 있게 될 것 같아요. 지금도 충분히 인상적이지만, 앞으로 몇 년 후에는 정말 전문 개발자 수준의 앱도 만들어낼 수 있을 것 같습니다.

## 결론

솔직히 말하면, 러버블 dev ai는 기대 이상이었어요. 물론 완벽하지는 않지만, 코딩을 모르는 일반인도 실제 작동하는 웹 애플리케이션을 만들 수 있다는 건 정말 혁신적입니다.

만약 간단한 웹앱이나 프로토타입을 빠르게 만들어야 하는 상황이라면 적극 추천드려요. 다만 복잡한 상용 서비스를 만들 계획이라면 아직은 전문 개발자의 도움이 필요할 것 같습니다.

그래도 이런 도구들이 계속 발전하면서 누구나 쉽게 자신만의 디지털 서비스를 만들 수 있는 시대가 오고 있다는 게 정말 흥미진진하네요. 여러분도 한번 시도해보시면 어떨까요?
